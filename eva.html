Code Evaluation Metrics Comparison

Metric Description Strengths Weaknesses
BLEU Measures similarity using n-grams. Widely used, easy to compute. May not capture semantic correctness.
ROUGE Focuses on recall with n-gram overlap. Useful for recall-focused tasks. Can overestimate quality with similar sequences.
CodeBLEU Considers code structure, syntax, and semantics. More suitable for code evaluation. More complex to implement.
Exact Match (EM) Percentage of cases with exact matches. Simple and interpretable. Too strict for minor variations.
Edit Distance Number of edits to match reference code. Granular measure of difference. Computationally expensive.
Semantic Similarity Measures functional equivalence. Focuses on functionality. Difficult to implement.
Human Evaluation Expert review of generated code. Assesses readability and maintainability. Time-consuming and subjective.
